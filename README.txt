En una olimpiada rusa se propuso el siguiente problema: Si escribimos todos los números enteros empezando por el uno, uno l lado del otro (o sea, 1234567891011121314), ¿qué dígito ocupa la posición 206

Definir una función que dado n calcule el elemento que ocupa la posición n en dicha sucesión. Por ejemplo,
elemento 5      = 5
elemento 10     = 1
elemento 11     = 0
elemento 206788 = 7
